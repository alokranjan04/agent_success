1. Product Overview

AgentOS is a real-time AI coaching platform for inbound call center agents. It acts as a silent AI co-pilot that reads conversation transcripts in real time and provides instructions, empathy alerts, and smart replies to ensure every customer interaction is professional and compliant.

  • Agent Skill Gaps: Turns new hires into experts on day one.
  • Script Compliance: Ensures identity verification and empathy are never skipped.
  • Reduced AHT: Smart suggestions reduce thinking time and silence durations.

3. Inbound Call Journey & AI Coaching

AgentOS guides agents through 5 critical stages of an inbound call:

  1. GREETING & VERIFICATION: Welcome warmly and confirm identity.
  2. ISSUE CAPTURE: Listen carefully and repeat back the issue.
  3. EMPATHY: Acknowledge and validate customer frustration.
  4. RESOLUTION: Offer a clear fix with concrete timelines.
  5. CLOSURE: Confirm satisfaction and invite final questions.

4. Feature List

  • Real-time Transcription: Instant voice-to-text for all participants.
  • AI Coaching Tags: Visual labels for critical moments (e.g., "Empathy Gap").
  • Escalation Risk: Dynamic meter predicting potential supervisor requests.
  • Smart Replies: Professionally crafted phrases to say or play (TTS).
  • Post-Call Summary: Full QA reports generated in seconds.

5. Knowledge Assist (RAG) New

The Knowledge Assist feature leverages Retrieval Augmented Generation to provide instant answers:

  • Modular Sidebar: A dedicated space for knowledge results.
  • Manual Search: Ad-hoc queries against your uploaded policies.
  • Document Indexing: Seamless PDF uploads via the Admin Panel.
  • Knowledge Snippets: Relevant chunks of text surfaced automatically based on the call context.

    8. Local Setup

    # Install dependencies
npm install

# Run development server
npm run dev

    Ensure your .env file contains the GEMINI_API_KEY and GOOGLE_APPLICATION_CREDENTIALS.

    9. GCP Deployment

    Deployment is handled via Container Registry and Cloud Run:

    gcloud builds submit --tag gcr.io/[PROJECT_ID]/agent-success
gcloud run deploy agent-success --image gcr.io/[PROJECT_ID]/agent-success --min-instances 1
